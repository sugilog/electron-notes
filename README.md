How To Use
-----------------------------

Be care, this repository is underconstruction.

1. Install node. (version=6 or over)
2. Clone this repository (recursive).
3. Run `npm install`.
4. Run `npm install -g gulp`.
5. Run `npm run setup`.
6. Run `npm start`.
